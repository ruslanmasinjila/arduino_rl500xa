GENERAL NOTES:

Arduino sketch pulseReader/pulseReader.ino is used to calibrate the transmitter.

Arduino sketch rcToPWM/rcToPWM.ino is used to convert RC signals to corresponding PWM signals.

Arduino sketches RC_control/RC_control.ino and RC_control/RC_control.ino are used to remotely steer the mobile platform RL-500-XA.

For further information, visit the following link: https://sites.google.com/site/ruslanmasinjila/rl_500_xa
